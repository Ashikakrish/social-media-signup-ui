# Facebook Signup Page UI

A responsive Facebook-inspired signup page interface developed using HTML and CSS for frontend practice.

## 🚀 Features

- Responsive layout
- Clean and modern UI
- Facebook-inspired design
- Beginner-friendly frontend project

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
facebook-signup-ui/
├── index.html
├── style.css
└── images/ (optional)
```

## ▶️ How to Run

1. Clone or download the repository.
2. Open `index.html` in your browser.
